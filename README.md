# HueHunt
โครงงานนี้เป็นส่วนหนึ่งของรายวิชา Computer Vision

## ส่วนที่จะพัฒนาเพิ่มเติม 
- [x] (เกม) การสุ่มชื่อสีและตำแหน่งกรอบสี่เหลี่ยม
- [x] (ฟลุค) การนับเวลาถอยหลัง 5 วิ
- [x] (แพร์) การนับคะแนนและการสรุปผลแพ้ชนะ

## สิ่งที่ควรคำนึบถึงเพิ่มเติม (ทำหลังฟังก์ชันหลักพัฒนาเสร็จแล้ว)
- [x] เช็คด้วยว่าสัดส่วนสีบนจอเป็นเท่าไหร่ของภาพทั้งหมด (ex. มากกว่า 50%) 
- [ ] ? ทำ Light Balance
- [ ] ? Candidate สี
- [x] 2 Mode เป็นยาก(มีสีที่แยกเฉดมากขึ้น ex. Blue Cyan Green) กับง่าย (Blue Green ปกติ)
